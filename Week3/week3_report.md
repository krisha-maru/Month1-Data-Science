## 3. Key Learnings

| Concept | Description | Key Properties | Practical Use |
|---|---|---|---|
| NumPy Arrays | Efficient structure for numerical data | Fast, homogeneous, supports vectorized operations | Numerical computation and machine learning workflows |
| NumPy Operations | Mathematical operations on arrays | Element-wise, efficient, loop-free | Bulk calculations on data |
| Broadcasting | Applying operations on arrays of different compatible shapes | Automatic expansion, concise syntax | Adding constants, scaling values |
| Pandas Series | One-dimensional labeled data structure | Indexed, flexible, easy to manipulate | Column-wise data handling |
| Pandas DataFrame | Two-dimensional tabular data structure | Row-column format, labeled axes, heterogeneous columns | Dataset storage and analysis |
| Indexing | Selecting rows and columns from data | `.loc`, `.iloc`, column selection | Data extraction and slicing |
| Filtering | Selecting rows based on conditions | Boolean masking | Data cleaning and subset creation |
| Missing Value Handling | Managing incomplete data | `dropna()`, `fillna()` | Data preprocessing |
| Grouping and Aggregation | Combining records to compute summaries | `groupby()`, aggregate functions like mean and sum | Reporting and insights generation |
