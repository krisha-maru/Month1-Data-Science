## 3. Key Learnings

| Concept | Description | Key Properties | Practical Use |
|---|---|---|---|
| Matplotlib | Core Python library for plotting graphs | Flexible, customizable, supports many chart types | Creating basic visualizations |
| Line Plot | Plot showing trends across ordered values | Useful for time or sequence-based data | Trend analysis |
| Bar Chart | Plot for comparing categories | Easy comparison across groups | Category-wise comparison |
| Scatter Plot | Plot showing relationship between two numerical variables | Helps detect correlation, clusters, outliers | Relationship analysis |
| Histogram | Plot showing distribution of a numerical variable | Uses bins, shows spread and concentration | Distribution analysis |
| Seaborn | Statistical data visualization library built on Matplotlib | High-level, attractive defaults, easier advanced plots | Exploratory data analysis |
| Heatmap | Visual representation of matrix values using color intensity | Useful for correlation matrices | Understanding feature relationships |
| Pairplot | Grid of pairwise relationships among numerical variables | Shows scatter plots and distributions together | Multivariable analysis |
| Dashboard Thinking | Organizing multiple plots for insight generation | Combines different chart perspectives | Client reporting and presentations |
